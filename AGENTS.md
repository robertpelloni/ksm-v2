# AGENTS.md

## General Instructions
- This project is K-Shoot MANIA v2 (v1 Remastered), a C++ implementation of the rhythm game K-Shoot MANIA.
- The goal is to fully reimplement v1 features in C++ with performance improvements and new features.
- Follow the style of the existing codebase.
- Use C++20 standard.
- The project uses OpenSiv3D.

## Documentation
- Maintain high-quality documentation in code and external files.
- Update `CHANGELOG.md` for every significant change.
- Update `DASHBOARD.md` when submodules change.

## Versioning
- The version number is stored in the `VERSION` file.
- When updating the version, update `VERSION`, `CHANGELOG.md`, and any other relevant files.

## Submodules
- Ensure all submodules are initialized and updated.
- Use `git submodule update --init --recursive` if needed.
